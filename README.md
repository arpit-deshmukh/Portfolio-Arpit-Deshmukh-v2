## 🎯 Features

### 🧭 **Modern Navigation Bar**
- Clean top navigation  
- Smooth scroll  
- Responsive on all devices  

### 🏠 **Home / Hero Section**
- Minimal hero design  
- Developer introduction  
- Clean typography  

### 👤 **About Section**
- Card-based layout  
- Personal introduction  
- Skills & strengths  
- Quick information block  

### 🎓 **Education Section**
- Clean education cards  
- Timeline-style structure  
- Fully customizable template  

### 💼 **Projects Section**
- Grid-based cards  
- Project images  
- Tech stack badges  
- GitHub & Live links  

### 🗂 **Portfolio Overview**
- Category cards (Projects, Activities, Research, Achievements, etc.)  
- Icon-based modern design  

### 🎯 **Activities & Experiences**
- Image + content cards  
- ITESA, Debate, workshops, clubs  
- Perfect for showcasing non-technical achievements  

### ✉️ **Contact Section**
- Clean contact card  
- Email, GitHub, LinkedIn, resume link  

### 🦶 **Minimal Footer**
- Simple & elegant  

### 🎞 **Animations**
- Framer Motion fade-in effects  
- Smooth, subtle, professional

## 🚀 Getting Started

1️⃣ Clone the project
```bash
git clone https://github.com/arpit-deshmukh/Portfolio-Arpit-Deshmukh-v2.git
cd client

2️⃣ Install dependencies
npm install

3️⃣ Start development server
npm run dev

4️⃣ Build for production
npm run build



```

 🌐 Deployment
Vercel
for vercel addd this script in vercel.json

```



"rewrites": [
        {
            "source": "/(.*)",
            "destination": "/index.html"
        }
    ]
}

---

```
